Sobre mi

¿Quién soy?

Hola, mi nombre es Mariano Aguilar, vivo en Baradero (Bs. As.-Argentina). Actualmente terminando la carrera de desarrollo web Fullstack de Apx, en último año de analista en informática en UNSADA y tercer año de licenciatura en informática en UNSADA. Me encanta la programación y cada día quiero aprender más, para mejorar mis habilidades y poder crear los proyectos que me proponga.

Habilidades

Html

Con mis conocimientos en HTML, puedo estructurar el contenido de las páginas y app web, como se observa en mis proyectos del portfolio, que junto a CSS y JS, llenan de funcionalidad y dan vida a estos proyectos.

Css

Con mis conocimientos actuales en CSS puedo replicar los diseños que me propongan y crear estilos muy variados, esto me permite que el diseño sea del agrado del cliente y los usuarios.

JavaScript

Tengo conocimientos sobre JavaScript, el cual me permiten crear web-apps y páginas web dinámicas.

TypeScript

Manejo del superset de JavaScript, que esencialmente añade tipos estáticos y objetos basados en clases, añadiendo un mayor control dotándolo de un typado fuerte para prevenir errores.

React

Tengo conocimientos en React, el cual es una librería que me permite crear componentes para reutilizarlos en toda la app, el cual mejora mucho el rendimiento de las web-apps, una de las librerías mas utilizadas en la actualidad.

Node.js

Con la tecnología Node.js, puedo crear el backend de las app-web y páginas webs dinámica, utilizando express, sql y la estructura de REST-API.

PostgreSQL

Conocimientos sobre SQL, en especial en Postgre, creación de bases de datos y consultas de SQL, también manejo de ORM como sequelize, abierto a aprender otros motores de BD y ORMS.

Next.js

Next.js es un potente framework web que se ha convertido en una elección popular entre los desarrolladores web para crear aplicaciones web modernas y eficientes gracias a su versatilidad.
